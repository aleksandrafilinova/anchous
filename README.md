## Тематика сайта, название

Игра "Wordle". Идея: Игрок должен угадать загаданное пятибуквенное слово за 6 попыток. После каждой попытки система даёт цветные подсказки о том, какие буквы угаданы и насколько верно они расположены.

## Цель и задачи сайта

Предоставить пользователям увлекательную и интеллектуально стимулирующую игру для развития лексического запаса и логического мышления..

## Целевая аудитория

Взрослые и подростки с хорошим знанием русского языка

## Содержание сайта

* начальная страница с правилами игры
* основная страница с игровым полем
* ...

## Функциональные возможности

* специальное текстовое поле для ввода пятибуквенного слова;

* автоматическая проверка длины слова (если введено меньше/больше 5 букв — предупреждение);

* подтверждение ввода по нажатию кнопки «Проверить» или клавиши Enter.

* подсветка букв после каждой попытки:

  * зелёный — буква на правильном месте;

  * жёлтый — буква есть в слове, но не на своём месте;

  * серый — буквы нет в загаданном слове;
